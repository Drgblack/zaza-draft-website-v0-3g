## The Complete AI Planning Framework

### Phase 1: Foundation Setting

**The Master Prompt Template:**

```
Create a [duration] lesson plan for [grade level] [subject] on [topic] that:
- Aligns with [specific standards]
- Includes [learning objectives]
- Accommodates [student needs/accommodations]
- Features [preferred teaching methods]
- Incorporates [available resources]
- Follows [school/district requirements]

Style: [Your teaching personality - engaging, structured, hands-on, etc.]
Assessment: [Formative/summative preferences]
Differentiation: [Specific student needs in your class]
```

**Sample Input:**

```
Create a 45-minute lesson plan for 4th grade Science on the Water Cycle that:
- Aligns with NGSS 5-ESS2-1
- Includes hands-on experiments and visual learning
- Accommodates 3 ELL students and 2 students with ADHD
- Features inquiry-based learning with small group work
- Incorporates available materials: tablets, basic lab supplies, outdoor space
- Follows our school's 5E model (Engage, Explore, Explain, Elaborate, Evaluate)

Style: Enthusiastic, hands-on, story-driven
Assessment: Exit tickets and practical demonstrations
Differentiation: Visual supports, movement breaks, peer partnerships
```

### Phase 2: Lesson Architecture

AI-generated lessons should include these components:

#### Essential Elements Checklist:

- [ ] **Hook/Warm-up** (5-7 minutes)
- [ ] **Learning Objectives** (student-friendly language)
- [ ] **Direct Instruction** (mini-lesson, 10-15 minutes)
- [ ] **Guided Practice** (scaffolded activities)
- [ ] **Independent Practice** (differentiated options)
- [ ] **Closure/Assessment** (exit strategy)
- [ ] **Extension Activities** (early finishers)
- [ ] **Homework/Follow-up** (optional)

#### Differentiation Matrix:

- **Advanced Learners:** Enrichment activities, leadership roles
- **Struggling Learners:** Modified materials, peer support
- **ELL Students:** Visual aids, vocabulary support
- **Special Needs:** Accommodations, alternative assessments

### Phase 3: Resource Generation

Use AI to create:

#### Teaching Materials

- **Presentation slides** with speaker notes
- **Graphic organizers** for student use
- **Vocabulary cards** with definitions and visuals
- **Reference sheets** for complex concepts

#### Assessment Tools

- **Formative assessment questions** throughout the lesson
- **Exit tickets** aligned to objectives
- **Rubrics** for project-based activities
- **Self-assessment checklists** for students

#### Parent Communication

- **Lesson summary** for family newsletters
- **Home extension activities** to reinforce learning
- **Vocabulary lists** for practice at home

## Subject-Specific AI Planning Strategies

### Mathematics Lessons

**Advanced Math Prompt:**

```
Design a math lesson that:
- Uses concrete-pictorial-abstract progression
- Includes multiple solution strategies
- Features real-world problem solving
- Incorporates math discourse and vocabulary
- Provides immediate error correction opportunities

Include: Number talks, guided practice problems, independent work stations, and assessment rubric.
```

**Sample Math Lesson Structure:**

1. **Number Talk** (5 min): Mental math warm-up
2. **Problem Introduction** (10 min): Real-world context
3. **Strategy Exploration** (15 min): Multiple approaches
4. **Practice & Apply** (12 min): Differentiated problems
5. **Math Discourse** (3 min): Share and justify solutions

### Language Arts Lessons

**ELA Power Prompt:**

```
Create a reading/writing lesson that:
- Integrates all literacy components (reading, writing, speaking, listening)
- Uses mentor texts and authentic literature
- Includes vocabulary in context
- Features student choice and voice
- Connects to students' lives and interests

Format: Workshop model with mini-lesson, work time, and sharing
```

### Science Lessons

**STEM Integration Prompt:**

```
Develop a science lesson that:
- Follows the 5E instructional model
- Includes hands-on investigation
- Integrates technology and data collection
- Connects to engineering and mathematics
- Features scientific discourse and writing

Include: Safety considerations, material lists, data collection sheets, and reflection questions.
```

### Social Studies Lessons

**Inquiry-Based Social Studies:**

```
Build a social studies lesson that:
- Uses primary and secondary sources
- Features historical thinking skills
- Includes perspective-taking activities
- Connects past to present
- Incorporates civic engagement

Structure: Document-based inquiry with multimedia resources
```

## Advanced AI Planning Techniques

### 1. Unit Planning with AI

Use AI to create coherent unit sequences:

```
Create a 3-week unit on [topic] that:
- Builds concepts progressively
- Includes formative and summative assessments
- Features culminating project or performance
- Connects to previous and future learning
- Includes cross-curricular connections

Provide: Daily lesson outlines, assessment calendar, resource list, and pacing guide.
```

### 2. Collaborative Planning

Generate lessons for team teaching:

```
Design co-teaching lessons that:
- Utilize both teachers' strengths
- Include parallel teaching, station teaching, and team teaching models
- Accommodate diverse learning needs
- Feature clear role definitions
- Include reflection and adjustment protocols
```

### 3. Project-Based Learning Design

Create comprehensive PBL units:

```
Develop a project-based learning experience that:
- Addresses authentic, real-world problems
- Includes community connections
- Features student choice and ownership
- Integrates multiple subjects
- Includes reflection and revision cycles

Include: Project timeline, checkpoint assessments, community partner information, and presentation guidelines.
```

## Troubleshooting Common Issues

### "The lessons feel too generic"

**Solution:** Enhance your prompts with specific student interests, local connections, and your teaching personality descriptors.

### "AI doesn't understand my students' needs"

**Solution:** Create detailed student profiles to include in your prompts, including learning differences, interests, and cultural backgrounds.

### "The pacing is always wrong"

**Solution:** Include timing preferences and your teaching style (fast-paced, reflective, discussion-heavy) in your prompts.

### "Activities don't match available resources"

**Solution:** Always specify your available materials, technology, and space constraints in initial prompts.
